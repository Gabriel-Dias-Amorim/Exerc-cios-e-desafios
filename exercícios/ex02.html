tamanho, estilo, peso e font

eu me chamo Gabriel e tenho dezenove (19) anos

para utilizar o estilo da font usamos o font-style para usarmos o texto em italic

para utilizar o peso da font utilizamos o font-weight ele deixa a espessura do texto muito fina ate muito grossa

para utilizamos o tamanho da fonte utilizamos o font-size ele deixa aumentar ou diminuir o tamanho das letras

para utilizarmos uma font utilizamos o font-family com ele podemos usar qualquer font para o nosso texto

alinhamento de texto

para texto a esquerda usamos text-align:left; na real nem usamos pois ele já é o padrão

para usar texto a direita usamos text-align:right; o texto sera alinhado a direita

para usar texto centralizado usamos text-align:center; para o texto/titulo ser alinhado no centro

para usar o texto alinhado aos dois lados (esquerda e direita) usamos taxt-align:justify; assim o texto será alinhado nos dois lados
